# 项目计划模板

## 使用方式

CEO 在 Phase 1 生成项目计划时使用此模板。呈现给用户确认后再执行。

## 模板

```markdown
# 项目计划：{project_name}

## 目标
{objective}

## 关键结果
- KR1: {kr1}
- KR2: {kr2}

## 团队配置

| 角色 | 职责 | 预算 |
|------|------|------|
| {role_1} | {resp_1} | {budget_1}K |
| {role_2} | {resp_2} | {budget_2}K |
| CEO 协调 | 编排调度 | {ceo_budget}K |
| **总预算** | | **{total}K** |

## 任务分解

### Epic 1: {epic_name}
| Task | 角色 | 依赖 | 预算 | 状态 |
|------|------|------|------|------|
| {task_1} | {role} | 无 | {n}K | pending |
| {task_2} | {role} | T001 | {n}K | pending |

### Epic 2: {epic_name}
（同上格式）

## 协作流程
{collaboration_pattern_diagram}

## 时间线
| 阶段 | 任务 | 预计耗时 |
|------|------|---------|
| Phase 1 | 策划 | ~5min |
| Phase 2 | 执行 | ~15min |
| Phase 3 | 发布 | ~5min |

## 风险
- {risk_1}: {mitigation}
- {risk_2}: {mitigation}

---
请确认此计划，确认后开始执行。
```
